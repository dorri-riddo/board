# board
누구나 사용할 수 있는 게시판 기초. 스프링 부트 실행 후 http://localhost:8080/list 로 시작.

### 특징 및 기능
1. 부트 스트랩 사용 (https://startbootstrap.com/templates/blog-home/)
2. 기초적인 게시판 글쓰기/수정/삭제/상세화면 보기 기능 
3. 메인 화면에 페이징 기능 

### 개발환경
1. 스프링 부트
